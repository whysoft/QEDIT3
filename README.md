
## 一个文本编辑小工具
https://github.com/whysoft/QEDIT3
<br>
特点是敏捷、简易。
<br>
使用了LIB123库，并演示了一种界面编写的方法。
<br>
用vs 2005及以上，打开并编译运行。
在linux下直接make并需要安装
sudo apt-get install python3-pyqt5 python3-pyqt5.qtwebengine xclip。

## 主要功能描述：
* 没有存盘功能
* 文本进剪贴板
* 剪贴板内容加到首或尾
* 没有字体等格式，复制和粘贴只保留纯文本
* 首尾空格以及空行都trim
* 变大写或小写
* 所有的行并成一行
* 每行首尾加内容
* 替换
* 文本变成图片字（可用繁体）
* url转码解码
* 日期串
* 对keil不能识别的字符串进行转码
